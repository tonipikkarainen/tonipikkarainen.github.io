7 Pitkittäinen aaltoliike

Kun ajat ohjelman huomaat, että koodi luo ruutuun yksittäisiä viivoja, joiden on tarkoitus esittää jousta. Huomaat ehkä myös, että voit liikkuttaa vasemman puoleisinta viivaa raahaamalla sitä hiirellä. Tehtävänäsi on saada muutkin viivat heilahtalemaan edestakaisin (x-suunnassa!), kun heilutat ensimmäistä viivaa. Eli tavoite on simuloida pitkittäistä aaltoliikettä jousessa. Katso apua tehtävästä 6 poikittainen aaltoliike.

Vinkki: sinun täytyy luoda jouselle jousivakio ja massa yksittäisille viivoille. Lisäksi sinun täytyy luoda silmukka, jossa saat viivat liikkumaan edellisen viivan liikkeen perusteella.

Esim. tällaiset rivit lisäämällä saat jousen värähtelemään pitkittäin.

{% highlight python %} k = 200 # N/m massa = 0.1 # kg while (True): rate(500) for i in range(1,len(viivat)): F_1 = -k*((viivat[i].pos.x-viivat[i-1].pos.x)-4)-10*viivat[i].nopeus viivat[i].nopeus = viivat[i].nopeus + F_1/massa*dt viivat[i].pos = viivat[i].pos + vector(viivat[i].nopeus,0,0)*dt t = t + dt {% endhighlight %}
